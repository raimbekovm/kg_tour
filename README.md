# The Last Caravan

### 10-Day Ecotourism Route Across Kyrgyzstan

> *From Silk Road Caravanserais to Celestial Mountains*

**[View Live Site](https://raimbekovm.github.io/kg_tour/)**

---

## About

An interactive single-page website presenting a 10-day ecotourism route across Kyrgyzstan — from Bishkek to Osh through alpine gorges, high-altitude lakes, ancient Silk Road monuments, and nomadic communities.

Built as part of the AUCA Ecotourism Route Development course (February 2026).

## Route Overview

| Day | Location | Highlights |
|-----|----------|------------|
| 1 | Bishkek | Ala-Too Square, Osh Bazaar, beshbarmak dinner |
| 2 | Ala-Archa & Supara | Ak-Say waterfall trek, eagle hunting show, felt workshop |
| 3 | Burana → Cholpon-Ata | 11th-c. minaret, Bronze Age petroglyphs, Issyk-Kul |
| 4 | Three Gorges → Karakol | Grigorievskoye, Semyonovskoye, Kyrchyn, Dungan mosque |
| 5 | Jeti-Ögüz → Skazka → Barskoon | Seven Bulls, Martian canyon, Gagarin waterfall |
| 6 | Kochkor → Son-Kul | Felt workshop, yurt camp at 3,016m, stargazing |
| 7 | Son-Kul → Tash-Rabat | Horseback ride, Silk Road caravanserai at 3,200m |
| 8 | Kazarman → Arslanbob | Two mountain passes, Uzbek homestay |
| 9 | Arslanbob → Uzgen → Osh | Walnut forest trek, UNESCO sites, Osh Bazaar |
| 10 | Osh | Ancient bazaar, souvenirs, departure |

## Site Sections

- **Hero** — fullscreen intro with route tagline
- **About** — key stats (10 days, 4 ethnic communities, 0 internal flights)
- **Day-by-Day Itinerary** — timeline with expandable hourly schedules
- **Photo Gallery** — real photographs from the route
- **Interactive Map** — Leaflet.js with 20 markers and route polyline
- **Culinary Journey** — 6 signature dishes from 4 cultures
- **Activities Checklist** — 5 pillars of ecotourism coverage
- **Elevation Profile** — SVG chart from 800m to 3,200m
- **Practical Info** — season, packing, budget, connectivity
- **Sustainability** — per-person environmental impact data

## Tech Stack

- Pure HTML / CSS / JavaScript (single file)
- [Leaflet.js](https://leafletjs.com/) + OpenStreetMap
- [Google Fonts](https://fonts.google.com/) (DM Serif Display, Inter)
- Intersection Observer API for scroll animations
- Deployed via GitHub Pages
